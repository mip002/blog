数花（count）题解

· 2023-01-26 · # 题解

题干传送门

先说结论：若用一个数组 ccc 来存数 i(i>1)i(i>1)i(i>1) 中 000 的个数，则：

  • 当 iii 为偶数， ci=ci2+1c_i=c_{\frac{i}{2}}+1ci​=c2i​​+1 ；
  • 当 iii 为奇数， ci=c⌊i2⌋c_i=c_{\lfloor\frac{i}{2}\rfloor}ci​=c⌊2i​⌋​ 。

举个例子，当 i=6i=6i=6 ，其二进制表示为 (110)2(110)_2(110)2​ 。 i2\frac{i}{2}2i​ 即为 333 ，二进制表示为 (11)2(11)_2(11)2​ ，相当于将 nnn 的二进制表示去掉最后一位 。而去掉的这一位刚好为 000 ，因此要在 ci2c_{\frac{i}{2}}c2i​​ 的基础上加上 111 。奇数同理。

这样以后，就可以 O(n)O(n)O(n) 处理出 10710^{7}107 内的 ccc 。

最后，再前缀和预处理一下 ccc 就可以 O(1)O(1)O(1) 询问了。

代码

#include<bits/stdc++.h>
using namespace std;
const int maxn=10000001;
int c[maxn];
int q[maxn];  // 前缀和
int n,m,l,r;
void init()
{
    c[0]=1,c[1]=0;
    if(n==0) q[0]=0,q[1]=1;
    else if(n==1) q[0]=q[1]=1;
    else q[0]=q[1]=0;
    for(int i=2;i<maxn;i++)
    {
        if(i&1) c[i]=c[i>>1];  //当 i 是奇数，c[i]=c[i/2]
        else c[i]=c[i>>1]+1;	//当 i 是偶数，c[i]=c[i/2]+1
        q[i]=q[i-1];
        if(c[i]==n) ++q[i];  
    }
}
int main()
{
    scanf("%d%d",&n,&m);
    init();
    while(m--)
    {
        scanf("%d%d",&l,&r);
        if(l==0) printf("%d\n",q[r]);
        else printf("%d\n",q[r]-q[l-1]);
    }
    return 0;
}
