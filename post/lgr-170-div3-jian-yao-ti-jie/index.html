T4

还是手玩几个样例，发现对于每次操作，aia_iai​ 只有以下几种变化方式：

  • 当 aia_iai​ 在序列 aaa 中仅出现一次，那么 ai=min⁡(ai,全局mex)a_i=\min(a_i,全局 \mathrm{ mex})ai​=min(ai​,全局mex)
  • 当 aia_iai​ 出现多次，那么 ai=全局mexa_i=全局 \mathrm{mex}ai​=全局mex

并且在进行若干次操作后，序列 aaa 中全局 mex\mathrm{mex}mex 为 max⁡{ai}+1\max\{a_i\}+1max{ai​}+1， 且要么所有的数互不相同，要么只有值为 mex−1\mathrm{mex}-1mex−1 的数有多个。如果为前者，则序列 aaa 在之后的操作中不会改变；若为后者，则序列 aaa 在一次操作后，所有值为 mex−1\mathrm{mex}-1mex−1 的数变为 mex\mathrm{mex}mex，其它元素不变，再进行一次操作后，又变为原样。

T5

我的构造方法：

  • 如果排列 ppp 中 111 的旁边既有 222 又有 333，则 aaa 中与 111 对应的位置放 444，与 333 对应的位置放 222，其它地方都放 111
  • 如果 111 的旁边有 222 没有 333，则 111 的位置放 333，其他位置都放 111
  • 否则，在 111 的位置放 222，其它位置都放 111

正确性显然。
