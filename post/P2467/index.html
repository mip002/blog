[SDOI2010] 地精部落题解

· 2023-10-28 · # 动态规划 # 题解

传送门

简要题意：求长为 nnn 的排列的波动序列的个数。

这道题的为什么是蓝题？感觉思维难度至少紫啊

考虑 DP，设 fi,jf_{i,j}fi,j​ 表示长度为 iii 的排列中，不合法元素有 jjj 个的方案数，其中不合法元素指的是一个排列里既不是山峰也不是山谷的元素。显然我们最后要的答案即为 fn,0f_{n,0}fn,0​。下面考虑转移

可以考虑如何从 fif_ifi​ 推 fi+1f_{i+1}fi+1​。这其实相当于在长度为 iii 的排列中插入一个数 i+1i+1i+1，显然 i+1i+1i+1 就是是这个排列里最大的数了。容易发现，插入这个数后这个排列的不合法元素要么不变，要么加一，要么减一，那么我们分别分析这三种情况。

由于长为 iii 的排列有 i+1i+1i+1 个位置是可插入的，而前面已经插入了 2+j2+j2+j 个位置，所以剩下的 i−j−1i-j-1i−j−1 个位置都会使不合法元素数量加一，即：

fi+1,j+1=fi+1,j+1+fi,j×(i−j−1)f_{i+1,j+1}=f_{i+1,j+1}+f_{i,j}\times (i-j-1) fi+1,j+1​=fi+1,j+1​+fi,j​×(i−j−1)

推出了转移方程这道题也就解决了，实现时注意用滚动数组即可。

Code

#include <bits/stdc++.h>
using namespace std;
#define endl "\n"
typedef long long ll;
const int maxn = 4205;
int n, mod;
int f[2][maxn];
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(0), cout.tie(0);
    cin >> n >> mod;
    f[1][0] = 4, f[1][1] = 2;
    for (int i = 3; i < n; i++)
    {
        int nxt = ((i + 1) & 1), now = (i & 1);
        for (int j = 0; j <= i - 1; j++)
            f[nxt][j] = 0;
        for (int j = 0; j <= i - 2; j++)
        {
            f[nxt][j] = (f[nxt][j] + 1ll * f[now][j] * 2 % mod) % mod;
            if (j)
                f[nxt][j - 1] = (f[nxt][j - 1] + 1ll * f[now][j] * j % mod) % mod;
            f[nxt][j + 1] = (f[nxt][j + 1] + 1ll * f[now][j] * (i - j - 1) % mod) % mod;
        }
    }
    cout << f[n & 1][0] << endl;
    return 0;
}
