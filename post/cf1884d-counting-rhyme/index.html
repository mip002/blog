CF1884D Counting Rhyme

· 2023-11-05 · # 动态规划 # 题解

题目传送门

题目翻译：

给定长度为 nnn 的序列 aaa。对于 1≤i<j≤n1\leq i<j\leq n1≤i<j≤n，若不存在 k∈[1,n]k\in[1,n]k∈[1,n] 使得 ak∣aia_k\mid a_iak​∣ai​ 且 ak∣aja_k\mid a_jak​∣aj​ 那么 (i,j)(i,j)(i,j) 是好的。求出好的数对数量。ai,n≤106a_i,n\leq 10^6ai​,n≤106。

注意到若 ak∣aia_k\mid a_iak​∣ai​ 且 ak∣aja_k\mid a_jak​∣aj​，那么 aka_kak​ 一定也能整除 gcd⁡(ai,aj)\gcd(a_i,a_j)gcd(ai​,aj​)。那么我们可以通过枚举 ggg，找出最大公约数为 ggg 的数对的个数 fgf_gfg​，最后答案即为 ∑fg[∀1≤k≤n,ak∤g]\sum f_g[\forall 1\leq k\leq n,a_k\nmid g]∑fg​[∀1≤k≤n,ak​∤g]。

下面考虑计算 fgf_gfg​。这里我们用 cntxcnt_xcntx​ 表示数字 xxx 在序列 aaa 中出现的次数，用 sumsumsum 表示能被 ggg 整除的数的个数。显然，sum=cntg+cnt2g+⋯+cntkg(kg≤n)sum=cnt_g+cnt_{2g}+\cdots+cnt_{kg}(kg\leq n)sum=cntg​+cnt2g​+⋯+cntkg​(kg≤n)。由此易知，所有最大公约数能被 ggg 整除的数对个数为 sum×(sum−1)2\frac{sum\times (sum-1)}{2}2sum×(sum−1)​，接下来还需要减去最大公约数大于 ggg 的情况，其实就是 f2g+f3g+⋯+fkg(kg≤n)f_{2g}+f_{3g}+\cdots+f_{kg}(kg\leq n)f2g​+f3g​+⋯+fkg​(kg≤n)。即：

fg=sum×(sum−1)2−f2g−⋯−fkgf_g=\frac{sum\times (sum-1)}{2}-f_{2g}-\cdots-f_{kg} fg​=2sum×(sum−1)​−f2g​−⋯−fkg​

注意到这个过程比较像埃氏筛，所以时间复杂度应该是 O(nlog⁡log⁡n)O(n\log\log n)O(nloglogn) 。但是官方题解写的是 O(nlog⁡n)O(n\log n)O(nlogn)，我也不知道到底哪个是对的，反正能过就行了。

代码

记得开 long long 。

#include <bits/stdc++.h>
using namespace std;
#define endl "\n"
typedef long long ll;
#define int long long
int T;
const int maxn = 1e6 + 5;
int n;
int a[maxn], cnt[maxn], vis[maxn];
int f[maxn];
void Solve()
{
    cin >> n;
    for (int i = 1; i <= n; i++)
        f[i] = cnt[i] = vis[i] = 0;
    for (int i = 1; i <= n; i++)
        cin >> a[i];
    for (int i = 1; i <= n; i++)
        cnt[a[i]]++;
    for (int i = 1; i <= n; i++)
        vis[a[i]] = 1;
    for (int i = n; i >= 1; i--)
    {
        int sum = 0;
        for (int j = i; j <= n; j += i)
        {
            sum += cnt[j];
            f[i] -= f[j];
            if (vis[i])
                vis[j] = 1;
        }
        f[i] += sum * (sum - 1) / 2;
    }
    int ans = 0;
    for (int i = 1; i <= n; i++)
        if (!vis[i])
            ans += f[i];
    cout << ans << endl;
}
signed main()
{
    ios::sync_with_stdio(false);
    cin.tie(0), cout.tie(0);
    cin >> T;
    while (T--)
    {
        Solve();
    }
    return 0;
}
